# Gençtek Blog

Gençlerin teknoloji dünyasında gelişmesine yardımcı olmak için hazırlanmış, HTML ve CSS ile yapılmış basit bir blog sitesi.

## 📋 Proje Detayları

- **Açıklama**: Gençtek tanıtım bloğu
- **Teknoloji**: HTML5, CSS3
- **Sayfalar**: 4 (Anasayfa, Blog, Hakkımda, İletişim)
- **Tasarım**: Responsive (Mobil uyumlu)

## 📁 Dosya Yapısı

```
Genctek-blog/
├── index.html          # Anasayfa
├── blog.html           # Blog yazıları sayfası
├── about.html          # Hakkımda sayfası
├── contact.html        # İletişim sayfası
├── style.css           # Tüm sayfalar için CSS stilleri
└── README.md           # Bu dosya
```

## 🎨 Özellikler

✅ **4 Sayfaya Sahip:**
- **Anasayfa (index.html)**: Blog tanıtımı ve öne çıkan yazılar
- **Blog (blog.html)**: Tüm blog yazıları
- **Hakkımda (about.html)**: Gençtek hakkında bilgiler
- **İletişim (contact.html)**: İletişim formu ve bilgileri

✅ **Responsive Tasarım**: Telefon, tablet ve bilgisayarlar için uyumlu

✅ **Modern CSS**: Gradient, animasyon, hover efektleri

✅ **Hızlı Yükleme**: Sadece HTML ve CSS, harici bağımlılık yok

## 🚀 Kullanım

1. Repository'yi klonlayın veya dosyaları indirin
2. `index.html` dosyasını web tarayıcınızda açın
3. Tüm sayfaları ziyaret edebilirsiniz

## 📝 İçerik

Blog yazıları şu konuları kapsar:
- Web Geliştirmeye Başlangıç
- Mobil Uygulama Geliştirme Trendleri
- Yapay Zeka ve Teknoloji

## 🎯 Gelecek İyileştirmeler

- JavaScript ile dinamik içerik
- Arama fonksiyonu
- Kategori filtreleme
- Yorum sistemi
- SEO optimizasyonu

## 📧 İletişim

Herhangi bir sorusu veya önerisi için contact.html sayfasındaki formu kullanabilirsiniz.
